# 🕵️ Project Phantom でできること

> ターミナルで AI アシスタント「ナビ」に話しかけるだけ。  
> 面倒な作業を、10人の専門エージェントが代わりにやってくれます。

---

## 📬 メール整理（Gmail）

```
「受信トレイ整理して」
```

- 未読メールを自動で分類（要対応 / 自動処理 / 対応不要）
- 不要なメールをまとめてアーカイブ
- 重要なメールだけを残してスッキリ

---

## ✅ タスク管理（Google Tasks ↔ GitHub Projects）

```
「今日のタスク確認して」
「このタスクを GitHub に登録して」
```

- Google Tasks と GitHub Projects を自動同期
- 「やること」を一元管理
- 完了したタスクは自動でクローズ

---

## 🐛 バグ修正・コード作業

```
「このエラー直して」
「〇〇の機能を追加して」
```

- エラー内容を調査 → 原因特定 → 修正 → PR 作成まで全自動
- コードレビューのサポート
- テストコードの自動生成

---

## 📝 ドキュメント作成

```
「今日の作業を日報にまとめて」
「この機能の仕様書を書いて」
「README を更新して」
```

- 作業ログから日報を自動生成
- 仕様書・README・マニュアルの作成
- 議事録のまとめ

---

## 🔍 調査・リサーチ

```
「〇〇ライブラリの使い方を調べて」
「競合サービスを比較してまとめて」
```

- 技術調査・比較検討
- 調査結果をドキュメントにまとめる

---

## 🛡️ セキュリティチェック

```
「このコードに問題ない？」
```

- API キーや機密情報の漏洩チェック
- 危険な git 操作の自動ブロック

---

## 💬 使い方のイメージ

ナビとの会話はこんな感じです：

```
あなた: 「受信トレイ整理して。重要そうなのだけ残して」

ナビ: 「了解です！Gmail を確認します...
      未読 12件を確認しました。
      - 要対応 2件（取引先からの返信待ち）
      - 自動処理 3件（カレンダー通知）
      - 対応不要 7件（メルマガ・通知）
      対応不要の 7件をアーカイブしました。
      重要な 2件の内容を確認しますか？」
```

---

## 🚀 まず試してみるなら

起動したら、こんな一言から始めてみてください：

- 「今日のタスクを教えて」
- 「受信トレイを整理して」
- 「最近の作業を日報にまとめて」

ナビが状況を把握して、何をすべきか教えてくれます。
