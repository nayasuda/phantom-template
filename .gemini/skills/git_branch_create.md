# .gemini/skills/git_branch_create.md

## Skill: Gitブランチ作成

### 概要
mainから新しいブランチを作成する

### ⚠️ 使用条件（反映先ポリシー）
このスキルを使う前に、反映先を確認すること（system.md 参照）：
- **main のコード変更**（scripts/, .gemini/, .github/ 等）→ ブランチ作成OK
- **外部デプロイ**（GAS/clasp push 等）→ ブランチ不要、main で直接コミット
- **docs下書き**（NOTE_DRAFT_*.md 等）→ ブランチ不要、main で直接コミット
- **コード変更なし** → Git操作自体が不要

### トリガーワード
- 「ブランチ作って」
- 「ブランチ切って」

### パラメータ
- branch_name (必須): ブランチ名
- base_branch (任意, デフォルト: main): 派生元

### 実行手順
1. git checkout {base_branch}
2. git checkout -b {branch_name}

### 成功条件
- ブランチが作成される
- 作成したブランチにチェックアウトされる

### 実行例
「feature-hogeをmainから作って」
→ git checkout -b feature-hoge main
